# KnowledgeCivic Commons

This repository contains public knowledge packs: short, source-based introductions to questions that affect civic life. Each pack distinguishes established facts, interpretation, uncertainty, and possible action.

## Knowledge pack format

1. The question in one sentence.
2. Why it matters.
3. Key concepts.
4. What the best available evidence says.
5. What remains uncertain or contested.
6. How to check the claim.
7. What a resident, learner, or community can do next.
8. Sources, licenses, and revision date.
